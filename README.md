# Universal File Optimizer (UFO)

A Python-based universal file optimizer capable of reducing the size of PDFs, Microsoft Office documents, images, and text files while preserving the original file format whenever possible.

## Features

- PDF optimization using Ghostscript and PyMuPDF
- Word (.docx) optimization
- Excel (.xlsx) optimization
- PowerPoint (.pptx) optimization
- Image optimization
- JSON/XML/CSV/TXT optimization
- Batch processing
- Google Colab compatible
- Automatic download of optimized files
- Keeps the original file if no reduction is achieved
